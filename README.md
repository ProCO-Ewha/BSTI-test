# ⚔️ BSTI ⚔️
블랙서바이벌 캐릭터 테스트 웹사이트입니다.

나와 플레이 성향이 가장 어울리는 **블랙서바이벌 캐릭터**는?
캐릭터 유형으로 확인해보세요!

```
cd frontend/
npm install
npm start
```
